Conteúdo

Este é o conteúdo da caixa. No modelo de caixas do CSS, o conteúdo é a parte interna onde o texto ou elementos aparecem.

Padding

O padding é o espaço interno entre o conteúdo e a borda da caixa. Neste projeto usamos padding: 25px.

Border

A borda envolve o padding e o conteúdo. Aqui usamos border: 3px solid para destacar cada seção.

Margin

A margem é o espaço externo entre uma caixa e outra. Usamos margin-bottom para separar as seções.
